# DBeaver破解

## 安装前准备

1. 下载DBeaver
    [Download DBeaver Edition](https://dbeaver.com/download/)

2. 下载java

   [Java Downloads | Oracle 中国](https://www.oracle.com/cn/java/technologies/downloads/)

3. 下载破解文件

## 开始安装

1. 安装DBeaver
2. 安装java

## 破解

1. 修改hosts

hosts文件路径：**C:\Windows\System32\drivers\etc\hosts**

```ini
127.0.0.1       dbeaver.com
```

2. 双击打开破解文件
